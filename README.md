# 🔐 تسجيل الدخول باستخدام Google (Firebase)

مشروع بسيط يوضح كيفية تسجيل الدخول باستخدام Google عبر Firebase Authentication.

## 🚀 المميزات
- تسجيل الدخول بحساب Google
- عرض اسم المستخدم والبريد والصورة
- تسجيل الخروج
- متابعة حالة تسجيل الدخول

## 🛠️ التقنيات المستخدمة
- HTML
- JavaScript
- Firebase Authentication

## ⚙️ الإعداد

1. أنشئ مشروع في Firebase
2. فعل Google Authentication
3. ضع بياناتك هنا في الكود:

```js
const firebaseConfig = {
  apiKey: "YOUR_API_KEY",
  authDomain: "YOUR_PROJECT.firebaseapp.com",
  projectId: "YOUR_PROJECT_ID",
};
